이제 이미지를 넣어보자 이미지는 img tag를 쓰는데 만으로는
이미지가 들어가지 않고 src 라는 source 의 줄임 tag 를 함께 써서
불러오고자 하는 웹 이미지 주소값을 넣어 삽입할 수 있다
그리고 불러온 이미지가 너무 클 때는 width 라는 tag 를 100% 값으로
같이 써서 웹화면에 맞춰 삽입해주면 된다.

부모(parent) 와 자식(child) 관계 tag 를 알아보자.
대표적인 것이 목록을 표현할 때 쓰는 List 의 li tag 인데 현재 공부 중인 화면의
제일 위에 써보면 목차 1. 2. 3. 을 넣고 li tag 로 작업해주면 된다
여기서 li tag 는 자식 tag 로 부모 tag 를 갖게 되는데
그게 ul tag 로 목차와 다음 list 를 구분 지을 때 필요하다
예를 들어 처음 목록은 수업 목록
두번째 목록은 참여자 목록 같은 경우 등을 말한다
마지막으로 부모 tag 인 ul tag 밑에 자식 tag로서 하위 tag 개념으로
표시해주기 위해 tab key 를 눌러 안으로 들여쓰기를 해준다
한마디로 grouping 으로 묶어줄 때 쓰면 좋다.

목록이나 앞에 숫자로 numbering 를 해준 것이 있다면 앞에 붙는
숫자 자동완성 tag 인 ol tag 를 써서 중간에 고친게 있어도
바로바로 숫자가 그에 맞춰 자동적으로 바뀌게 해놓을 수 있다

ol tag 의 ol 은 ordered list 의 약자고 ul tag 의 ul 은 unordered List 의 약자다.

공통으로 추가하거나 공통으로 수정하고 싶은 tag 명이 있다면
ctrl key 를 누른 채로 해당 부분을 각각 클릭 클릭으로 고른 뒤
고칠 내용을 넣어주면 한번에 삽입 및 수정이 되게 된다.

추가적으로 표를 작성할 때 쓰는 tag 는 table tag 인데
부모/자식 tag 관계에서 이 tag 는 3대인 손자까지 같이 붙어 다니게 된다
밑에 예시를 보자

tag 로 구성된 문장이 모여 (웹)페이지가 되고 페이지가 모여 책이 된다
책에는 표지가 있고 저자가 표시된다 이렇게 책을 구성하듯 구조에 대해 알아보자
title tag 는 웹 페이지의 제목이면서 검색엔진에서 이 제목을 보고
찾아내는 경우가 많기 때문에 title tag 는 반드시 그리고 잘 적어주어야 한다
한글로 text 를 치고 표시하려면 컴퓨터 언어는 숫자 0과 1로 구성이 되어 있어서
utf-8 로 저장하고 불러들이고 읽어들어야지 제대로 표시된다
title tag 밑에 meta tag 를 넣고 utf-8 로 페이지를 읽어달라고 주문해주면 된다

웹 페이지를 구성할 때 제목 영역과 본문 내용의 영역을 구분해주는 것은 중요하다
본문을 구성하는 tag들은 body tag 를 써서 묶어주고
제목 영역은 head 라는 tag 를 써서 묶어준다
그리고 그 상위에는 모든 tag 를 묶는 html tag 를 삽입해준다
마지막으로 이 페이지는 html 로 구성되어 있다는 표시로
!doctype html 을 넣어주면 된다.

웹 페이지 내 외부링크를 걸어줄 때는 정박할 때 쓰는 "닻" 이라는 뜻의 anchor 의
a tag 를 써주는데 링크를 걸어줄 곳에 a 에 hyper text 를 뜻하는 h
그리고 속성을 뜻하는 reference 의 ref 를 붙여 주소값을 넣어주면 된다
그리고 그 링크를 열 때 새창에서 열리게 하려면 target=_blank" 를 함께
적어주면 되고 링크에 마우스오버 시 tool tip 으로 미리 알려주려면
title="설명" 을 넣어주면 된다
